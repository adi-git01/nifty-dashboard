## Setup

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `streamlit run main.py`

## Email Notifications

To receive weekly reports:
1. Go to the Return Tracker tab
2. Configure your Gmail address and App Password
3. Get your App Password from [Google Account Settings](https://myaccount.google.com/apppasswords)
